Default controls

Controls can be changed in the Settings menu. The defaults are:

KeyboardGamepadEffect
Arrows or navigation
/ select/ok
/ cancel/back
/ game details
/ filter and search
or L1/LBprevious collection
or R1/RBnext collection
Page Up L2/LTprevious page of games
Page Down R2/RTnext page of games

In addition, if you have a keyboard you may use + [letter] to jump to the first game starting with that letter.
