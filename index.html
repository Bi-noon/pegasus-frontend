    APT repository

    # Add the repository
curl https://bintray.com/user/downloadSubjectPublicKey?username=bintray | sudo apt-key add -
sudo add-apt-repository 'deb https://dl.bintray.com/mmatyas/deb testing main'

# Update and install
sudo apt update
sudo apt install pegasus-frontend

    System requirements

    • A 64-bit linux distro not older than Ubuntu 14.04 or Debian Wheezy (to be more specific, Pegasus needs glibc-2.17 + glibcxx-3.4.18, aka G++ 4.8 or later)
    • The main dependencies are Fontconfig and GStreamer 1.0 with its plugins. On Ubuntu/Debian, you can install them with sudo apt-get install fontconfig gstreamer1.0-alsa gstreamer1.0-libav gstreamer1.0-plugins-good (though they tend to come preinstalled on most desktop distros).
    • While the releases are designed to be fairly portable, they're not guaranteed to work on every distribution. If necessary, you can build your own version following the documentation.

    Installation

    Just extract and run.

    RetroPie repository

    Pegasus is also available from the RetroPie setup script, under the Experimenal section. It will install the latest version for your device type.

    System requirements

    • A Debian Jessie or later
    • Fontconfig and GStreamer 1.0 with its plugins. You can install them with sudo apt-get install fontconfig gstreamer1.0-alsa gstreamer1.0-libav gstreamer1.0-plugins-good
    • Optional: PolicyKit is required for restarting the device as a regular user (sudo apt-get install policykit-1)
    • No desktop environment is required.

    Installation

    Just extract and run the version for your device (you'll see Illegal instruction errors when you try to run the wrong release). Note that the system may not allow running programs from USB sticks.

    System requirements

    • Android 4.4 (KitKat) or later

    Installation

    First you'll need to enable installing APK packages, which can be done under Settings → Security → Unknown Sources, turned on. Then open the APK file with any file manager app.
    Note that if you have an active full screen app (eg. screen shade), Android might require you to disable it for the time of the installation (for security reasons).
